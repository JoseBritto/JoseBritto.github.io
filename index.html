Projects ...

Some awesome projects that I worked on

Denaro

A GTK app to manage your personal finances.

  • Manage multiple accounts
  • Easily filter transactions
  • Easily repeat transactions
  • Transfer money between accounts
  • Export an account as a CSV file and import a CSV, OFX or QIF file

Moosic

A discord bot that can play music all day long for you and your friends to enjoy.

  • Searches for music by name
  • Supports custom playlists
  • Plays spotify playlists
  • Created using C# and .Net 7

Game of Life

Recreation of the cellular automaton "Game of Life" by John Conway.

  • 0 player game
  • Runs 100+ simulations per second
  • Created using C# and Unity

Flappy

My shot at recreating the classic game of "Flappy Bird".

  • Collision detection
  • Simple gravity & physics system
  • Keeps track of the high score
  • Created using C# and Unity
